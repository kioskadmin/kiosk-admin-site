## Everything you need to manage your fleet

<CardGrid stagger>
  <Card title="Live device overview" icon="laptop">
    Battery level, screen state, current URL, and online/offline history — updated in real time via MQTT or 30-second polling.
  </Card>
  <Card title="Remote control" icon="setting">
    Screen on/off, URL navigation, restart, screensaver, TTS, volume, JavaScript injection, D-pad remote, and more.
  </Card>
  <Card title="Three providers" icon="puzzle">
    Supports **Fully Kiosk Browser**, **FreeKiosk**, and **Fully Cloud** out of the box. The same UI adapts to each provider's capabilities. Adding a new provider takes a single class.
  </Card>
  <Card title="Embedded MQTT broker" icon="seti:config">
    Toggle "Embedded Broker" in Settings and devices connect directly — no Mosquitto or HiveMQ account needed. Or connect to any external broker. Falls back to 30-second HTTP polling.
  </Card>
  <Card title="MQTT device discovery" icon="magnifier">
    Unregistered devices broadcasting over MQTT appear as a banner in the dashboard. Add them in one click with fields pre-filled from the MQTT payload.
  </Card>
  <Card title="Cross-network devices" icon="right-arrow">
    Route commands through MQTT for devices behind NAT, on remote networks, or behind cellular — no port forwarding required.
  </Card>
  <Card title="Bulk actions" icon="list-format">
    Send a command to every device in a group with one click. Filter by tag, group, or status before dispatching.
  </Card>
  <Card title="Config templates" icon="document">
    Save device settings as a named template. Apply the same preset to dozens of devices in seconds.
  </Card>
  <Card title="Alerts & audit log" icon="warning">
    Email alerts for offline, low battery, and unplugged events. Every command is logged with user, device, and timestamp.
  </Card>
  <Card title="Zero-config default" icon="rocket">
    Runs on SQLite out of the box. Switch to PostgreSQL or MongoDB by setting a single environment variable.
  </Card>
</CardGrid>
